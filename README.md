# RecommenderSystem

This project is provides a Jupyter Notebook with a recommender engine for articles on the IBM Watson Studio platform. 

The recommendation engine includes following techniques:
* Rank Based Recommendations
* User-User Based Recommendations
* Content Based Recommendations
* Matrix Factorization


### Installation 

To run this the Jupyter Notebook a installation of Anaconda is recommendet.

[Install Anaconda](https://www.anaconda.com/products/individual#Downloads)

Alternatively you can also pip-install Jupyter-Notebook: 

[Install Jupyter without Anaconda](https://jupyter.org/install)

In both cases the following Python packages are required:
* Pands
* Numpy
* Matplotlib
* Pickle 
* sklearn
* re
* nltk
